# Розаден

*Чернильный Писчий. Кровавый демон Династии Скрипторов. Первый круг.*

---

Архивариус, контракт-маг, разведчик через чернила. Служит [[Локации/Домены/Домен Паймона/Семья Дакел|Дому Дакел]]. Кровь — чёрная как чернила, может оживлять написанное на бумаге.

## Краткий профиль

| | |
|---|---|
| **Раса** | Кровавый демон, Династия Скрипторов |
| **Круг** | 1 |
| **Главный талант** | Ожившие чернила *(Дэкар+Крод)* — маленькие шпионы из капли крови-чернил на бумаге |
| **Расовое** | Чернильная кровь, бумажная кожа, фотографическая память |
| **Дом** | [[Локации/Домены/Домен Паймона/Семья Дакел|Дакел]], домен [[Игромеханика/Владыки/Паймон|Паймона]] |
