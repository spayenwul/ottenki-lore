### Новые параметры для точности

Для максимальной гибкости введем несколько специализированных коэффициентов "Конструкции" (К) и один новый параметр.

*   `К_удар`: Эффективность нанесения ударного/рвущего урона (лапы, кулаки, челюсти).
*   `К_подъем`: Эффективность при поднятии тяжестей (биомеханика спины, рук).
*   `К_опора`: Устойчивость, сцепление с поверхностью (количество ног, центр тяжести).
*   **Новый параметр: Скорость/Рывок (V):** Множитель, отражающий, насколько быстро и резко выполняется действие. Неловкий толчок (V=0.8) менее эффективен, чем резкий удар с разбега (V=1.5).

### Формулы для разных действий

Вот как можно разбить и улучшить расчеты для разных ситуаций.

#### 1. Урон от удара (атака без оружия)

Эта формула заменит вашу основную. Она моделирует кинетическую энергию.

**Формула:** `Урон = (СМ * К_удар) * (1 + М / М_база) * V`

*   `СМ`: Сила Мышц.
*   `К_удар`: Коэффициент конструкции для удара.
*   `М`: Масса атакующего.
*   `М_база`: "Базовая масса". Это константа, равная массе среднего существа (например, 100 кг). Она нужна, чтобы превратить массу в удобный множитель. Существо с массой 100 кг получит множитель `(1 + 100/100) = 2`. Существо с массой 400 кг получит `(1 + 400/100) = 5`.
*   `V`: Множитель скорости/резкости атаки (по умолчанию 1.0).

**Сравнение на примере (V=1.0):**

*   **Самый сильный человек:**
    *   СМ=100, М=140 кг, К_удар=1.0 (человеческий кулак)
    *   `Урон = (100 * 1.0) * (1 + 140 / 100) = 100 * 2.4 = **240 ед. урона**`

*   **Медведь Гризли:**
    *   СМ=150, М=400 кг, К_удар=2.5 (мощная лапа с когтями)
    *   `Урон = (150 * 2.5) * (1 + 400 / 100) = 375 * 5 = **1875 ед. урона**`

**Что изменилось?** Разница стала колоссальной, и это реалистично. Удар медведя — это не просто сильный удар. Это удар, в который вложена огромная масса, и его лапа конструктивно создана для нанесения увечий. Формула это отражает идеально.

#### 2. Поднятие и переноска тяжестей

Здесь масса самого существа не так важна, как его мышечная сила и правильная биомеханика для подъема.

**Формула:** `Поднимаемый Вес (кг) = СМ * К_подъем * C_масштаб`

*   `СМ`: Сила Мышц.
*   `К_подъем`: Коэффициент конструкции для подъема.
*   `C_масштаб`: Константа масштабирования, чтобы привести результат к реалистичным килограммам. Подбирается один раз для всей системы. Допустим, `C_масштаб = 2.5`.

**Пример:**

*   **Самый сильный человек (пауэрлифтер):**
    *   СМ=100, К_подъем=2.0 (идеальная техника становой тяги)
    *   `Вес = 100 * 2.0 * 2.5 = **500 кг**` (соответствует мировым рекордам).

*   **Медведь Гризли:**
    *   СМ=150, К_подъем=1.5 (силен, но его тело не оптимизировано для подъема тяжестей с земли, как у человека)
    *   `Вес = 150 * 1.5 * 2.5 = **562.5 кг**` (сильнее человека, но не в разы, что тоже логично).

Эта формула позволяет создавать существ, которые при меньшей "общей" силе могут быть феноменальными грузчиками (муравьи, жуки) за счет огромного `К_подъем`.

#### 3. Толкание и Устойчивость (противодействие толчку)

Здесь нам нужны две противоборствующие величины: **Сила Толчка** одного существа и **Стабильность** другого.

**Формула Силы Толчка:** `Сила Толчка = (СМ * 0.5 + М * 0.5) * К_удар * V`
*   Мышцы и масса вкладываются в толчок примерно поровну.

**Формула Стабильности (Сопротивления):** `Стабильность = (М * 0.7 + СМ * 0.3) * К_опора`
*   Здесь масса важнее, она вас "якорит". Сила помогает упереться. `К_опора` — ключевой параметр.
    *   Человек на двух ногах: `К_опора = 1.0`
    *   Четвероногое животное (медведь): `К_опора = 2.0` (гораздо устойчивее)
    *   Существо с низкой посадкой (краб): `К_опора = 2.5`
    *   Дерево (если его толкать): `К_опора = 10.0`

**Сценарий: Человек толкает медведя.**

*   **Человек (Сила Толчка):**
    *   СМ=100, М=140, К_удар=1.0, V=1.2 (с разбега)
    *   `Сила Толчка = (100*0.5 + 140*0.5) * 1.0 * 1.2 = (50 + 70) * 1.2 = 120 * 1.2 = **144**`

*   **Медведь (Стабильность):**
    *   СМ=150, М=400, К_опора=2.0
    *   `Стабильность = (400*0.7 + 150*0.3) * 2.0 = (280 + 45) * 2.0 = 325 * 2.0 = **650**`

Результат: 144 против 650. Человек не сдвинет медведя с места, даже если тот не будет активно сопротивляться. Это идеально моделирует реальность.
