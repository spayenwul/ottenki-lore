Понял! Теперь логика ясна — **блочная структура** с материнской руной.

## Формулы

### Структура заклинания

```
Заклинание = Блок_1 + Блок_2 + ... + Блок_N

Блок = Материнская_руна + Придатки (младшие)
```

### СЗ блока

```
ДЗ_блока = ДЗ_материнской + Σ(ДЗ_придатков)

СЗ_блока_сырое = База_материнской × ДЗ_блока

СЗ_материнской = СЗ_блока_сырое × (1 - Налог_придатков)
СЗ_придатков = СЗ_блока_сырое × Налог_придатков
```

### Налог придатков

**Если материнская — старшая:**

```
Налог_придатков = 10% × Количество_младших
```

**Если материнская — средняя:**

```
Налог_придатков = 10% × Количество_младших
```

### СЗ заклинания

```
СЗ_заклинания = Σ(СЗ_блока_сырое) для всех блоков
```

---

## Пример 1: Старшая-матерь

Храс (база 50) + 3 младших

```
ДЗ_блока = 3 + 3 = 6
СЗ_блока_сырое = 50 × 6 = 300

Налог = 10% × 3 = 30%

СЗ_Храс = 300 × 0.7 = 210
СЗ_младших = 300 × 0.3 = 90 (делят между собой по 30)
```

---

## Пример 2: Два независимых блока

**Блок 1:** Храс (база 50) + 3 младших **Блок 2:** Шейр (база 20) + 1 младшая

```
Блок 1:
ДЗ = 3 + 3 = 6
СЗ_сырое = 50 × 6 = 300
Налог = 30%
СЗ_Храс = 210, СЗ_младших = 90

Блок 2:
ДЗ = 2 + 1 = 3
СЗ_сырое = 20 × 3 = 60
Налог = 10%
СЗ_Шейр = 54, СЗ_младшей = 6

Итого СЗ заклинания: 300 + 60 = 360
Общий ДЗ: 9
```

---

## Пример 3: Средняя-матерь

Шейр (база 20) — матерь + 5 младших + Храс (база 50) как придаток

```
ДЗ_блока = 2 + 5 + 3 = 10
СЗ_блока_сырое = 20 × 10 = 200

Налог = 50% (потому что средняя — матерь)

СЗ_Шейр = 200 × 0.5 = 100
СЗ_придатков = 200 × 0.5 = 100

Храс как придаток:
СЗ_Храс = 50 × 3 = 150 (свой ДЗ × своя база, независимо)
```
