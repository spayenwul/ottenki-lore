# Врата Жизни

Магический судоходный канал, соединяющий внешний океан и внутреннее Внутреннее море. Единственный проход для крупных судов между двумя водоёмами. Контролируется [[Элзия|Элзией]].

---

| Параметр             | Значение                                                           |     |
| -------------------- | ------------------------------------------------------------------ | --- |
| **Тип**              | Магический канал с системой фильтрации                             |     |
| **Расположение**     | Перешеек между океаном и Кровавым морем                            |     |
| **Функция**          | Судоходный проход + опреснение воды                                |     |
| **Создатели**        | [[Агнейр Делир]] (архитектура) + [[Филипп Салазар]] (руническая система) |     |

| **Источник энергии** | Пассивный сбор Творения с населения региона + сила Агнейра         |     |

## История создания

### Предпосылки
- Внутреннее море непригодно для питья (высокая солёность, минералы)
- [[Элзия]] зависела от импорта пресной воды
- Демон-патрон [[Агнейр Делир]] предложил грандиозный проект

### Проектирование (120–115 лет назад)
- **Агнейр:** архитектурные планы, демоническая кровь как «клей» для рун
- **[[Филипп Салазар]]:** оживление врат как живого организма. Друг Агнейра, прожил 35 лет

### Строительство (115–110 лет назад)
- ~30 000 рабочих (обе расы)
- 500 магов под руководством Филиппа Салазара
- Финансирование: [[Великая Двойная Империя|Империя Варала]]
- **Цена:** ~5 000 погибших (несчастные случаи, магические выбросы)

### Открытие (110 лет назад)
- Активация Агнейром. Филипп Салазар погиб при строительстве, став частью Врат
- Вода из океана потекла во внутреннее море
- Соль и железо отфильтровались — вода стала почти пресной

## Устройство

### Канал
- Соединяет внешний океан с внутренним Кровавым морем
- Глубина достаточна для прохода крупных торговых судов
- Стены укреплены магически обработанным камнем (добыт из Каменных Лесов Элзии — единственного доступного источника камня на островах)

### Система фильтрации
- Руническая сеть извлекает соль и железо из воды
- Побочный продукт: осадок с оксидом железа → используется в производстве [[Красная Гавань#Кровавая Керамика|Кровавой Керамики]]

### Источник энергии
- Пассивный сбор [[Суть|Сути]] с населения региона (идея Филиппа Салазара)
- Часть силы [[Агнейр Делир|Агнейра]] навсегда закреплена в конструкции
- После [[История Элзии#Великое Изгнание|Великого Изгнания]] Врата продолжают работать автономно

## Город Врата-над-Морем

Город, выросший вокруг Врат — крупнейший в [[Элзия|Элзии]].

| Параметр | Значение |
|----------|----------|
| **Население** | ~250 000 жителей |
| **Статус** | Де-факто столица Элзии |
| **Особенность** | Нейтральная зона (Центральный Остров) |

### Структура
- **Северная часть:** контролируется [[Дом Солнечного Горизонта|Домом Солнечного Горизонта]] (Евклиды)
- **Южная часть:** контролируется [[Клан Багровой Матери|Кланом Багровой Матери]] (Залиды)
- **Центральный Остров:** нейтральная зона, управляется [[Орден Двойного Пламени|Орденом Двойного Пламени]]
  - Посольства всех держав
  - Объединённая Стража Врат (50% Евклиды, 50% Залиды)

### Экономика
- Пошлины: 5–7% от стоимости груза
- Доход от торговли — основа экономики Элзии
- Через Врата идёт международная торговля всего региона

## Стратегическое значение

### Для Элзии
- Контроль над Вратами = экономическое могущество
- Нейтралитет защищён угрозой закрытия Врат
- Врата требуют энергии **обеих** рас — принудительный симбиоз

### Для региона
- Единственный судоходный проход в Внутреннее море
- Закрытие Врат = коллапс региональной торговли
- [[Империя Углеас]] стремится получить контроль над Вратами

### Исторические претензии
- [[Королевство Хакирия]]: ультиматум ~99 лет назад, требовали передачи контроля
- [[Империя Углеас]]: ультиматум ~2 недели назад, готовы к интервенции
