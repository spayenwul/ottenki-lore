# Политика Элзии

Политическая система [[Элзия|Элзии]] — формальный баланс между [[Евклиды и Залиды|Евклидами и Залидами]], фактически контролируемый олигархией «Пяти Столпов».

---

## Официальная структура власти

### Верховный Арбитр
- Глава государства, избирается Двойным Советом
- Ротация между расами каждые ~50 лет
- **Текущий:** Дал'Мирас Строгий (Залид, 65 лет, правит 20 лет)
- Полномочия формально широкие, реально ограничены Столпами

### Двойной Совет
| Орган | Состав | Функции |
|-------|--------|---------|
| **Совет Света** | 25 Евклидов | Религия, северные территории, внешняя торговля |
| **Совет Земли** | 25 Залидов | Производство, южные территории, оборона |

Каждый Совет имеет право вето на решения Арбитра.

### Объединённая Стража Врат
- 50% Евклиды, 50% Залиды
- Командование ротируется ежегодно
- Контролирует Нейтральную Зону (Центральный Остров)

## Пять Столпов (реальная власть)

### 1. Дом Солнечного Горизонта
| Параметр | Значение |
|----------|----------|
| **Раса** | Евклиды |
| **Глава** | Патриарх Иларион Сол'Кварис (78 лет) |
| **Контроль** | Северная половина Врат, храмы Фобоса, торговля с [[Королевство Хакирия\|Хакирией]] |
| **Ресурсы** | 40% земель севера, «Жреческая Гвардия» (1 000 воинов) |
| **Девиз** | «Свет освещает путь, золото прокладывает дорогу» |

### 2. Клан Багровой Матери
| Параметр | Значение |
|----------|----------|
| **Раса** | Залиды |
| **Глава** | Матриарх Эш'Нара Кровь-и-Глина (54 года) |
| **Контроль** | [[Красная Гавань]], южная половина Врат, производство керамики |
| **Ресурсы** | Монополия на «Кровавую Керамику», «Гильдия Кровавых Мастеров» |
| **Девиз** | «Кровь питает землю, земля рождает силу» |

**Характер Эш'Нары:** Жёсткая, мстительная. Её мать убита в «Ночи Разбитых Зеркал» (~55 лет назад).

### 3. Гильдия Морских Клинков
| Параметр | Значение |
|----------|----------|
| **Раса** | Смешанная (космополиты) |
| **Глава** | Адмирал Велиар Острый Парус (48 лет) |
| **Контроль** | Флот, контрабанда, пролив Врат |
| **Ресурсы** | Тайный союз с [[Русалы\|Русалами]], агентурная сеть [[Мелеефаты\|Мелеефатов]] |

**Характер Велиара:** Циник, жадный. Не верит в идеалы. Тайно купил особняк в [[Дэрия|Дэрии]] на случай краха.

**Альянсы:**
- **Русалы** (~48 лет назад): Гильдия платит 2% от грузов металлом; Русалы контролируют пролив
- **Мелеефаты** (~45 лет назад): Льготные пошлины в обмен на разведданные

### 4. Орден Двойного Пламени
| Параметр | Значение |
|----------|----------|
| **Раса** | Смешанная (нейтральная) |
| **Глава** | Верховный Маг [[Тейас Межсолнечный]] (62 года, полукровка) |
| **Контроль** | Маги Врат, [[Академия Сияющей Земли]] |
| **Позиция** | Последняя нейтральная сила |

**Статус:** [[Тейас Межсолнечный|Тейас]] ранен в результате покушения (~10 дней назад). Орден ослабевает — студенты делятся по расовому признаку.

**История:** Основан ~95 лет назад. Почитает [[Агнейр Делир|Агнейра]] как символ единства рас.

### 5. Культ Единого Света
| Параметр | Значение |
|----------|----------|
| **Раса** | Евклиды |
| **Глава** | Пророчица Сэйрин (70 лет) |
| **Идеология** | «Фобос — единственный свет. Залиды поклоняются тьме.» |
| **Численность** | ~50 000 фанатиков |

**Тезисы учения:**
1. Фобос — единственный истинный свет; Деймос — лишь отражение
2. Залиды поклоняются стихиям тьмы
3. Смешение рас — грех (бесплодие полукровок — кара)
4. Элзия должна быть «очищена»

**Связь с Домом:** Тайно финансируется Домом Солнечного Горизонта, но вышел из-под контроля.

## Международные отношения

### Империя Углеас
- Анклав во [[Врата-над-Морем]] с ~92 года назад
- ~5 000 «торговцев» (половина — шпионы)
- **Стратегия:** «Разделяй и властвуй» — финансируют обе стороны конфликта
- **Цель:** Ослабить Элзию → установить протекторат над Вратами

### Королевство Хакирия
- Торговый партнёр Евклидов
- Поставки продовольствия в обмен на доступ к Вратам
- Исторические претензии на контроль над Вратами

### Королевство Схабди
- Поставщики наёмников ([[Миелонцы]])
- Экспортёры качественного оружия
- Конфликт с [[Мелеефаты|Мелеефатами]] из Йорлина

---

## Текущий политический кризис

**Проблема:** После исчезновения Деймоса (~1.5 месяца назад) баланс разрушен.

| Сторона | Позиция |
|---------|---------|
| **Дом Солнечного Горизонта** | Требует передачи власти Евклидам |
| **Клан Багровой Матери** | Отказывается, готовится к обороне |
| **Культ Единого Света** | «Очищение» — геноцид Залидов |
| **Красные Горизонты** | Превентивный удар по Евклидам |
| **Арбитр Дал'Мирас** | Пытается медиировать, игнорируется |
| **Орден Двойного Пламени** | Глава ранен, нейтралитет под угрозой |
| **Империя Углеас** | Готова к интервенции «для защиты торговли» |
