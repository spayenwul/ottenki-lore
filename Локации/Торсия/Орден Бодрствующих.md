# Орден Бодрствующих

Военно-религиозный орден [[Торсия|Торсии]], воины которого **не спят** во время Зимнего Покоя, охраняя беззащитный народ [[Айни]].

---

## Ключевая информация

| Параметр           | Значение                                           |
| ------------------ | -------------------------------------------------- |
| **Статус**         | Военно-религиозный орден                           |
| **Численность**    | ~1 500 воинов                                      |
| **Подчинение**     | Хранитель Ветви Тени (Грозан Сумрачный)            |
| **Штаб-квартира**  | Застава Рассвета (граница Гнилых Топей)            |
| **Основан**        | ~800 лет назад (после Войны Пробуждения)           |
| **Командор**       | Грим Несонный (58 лет)                             |

---

## История

### Основание

Орден был создан после **Войны Пробуждения** (~800 лет назад), когда частичное «пробуждение в кошмар» [[Великое Древо Хол|Хола]] показало уязвимость народа во время спячки.

**Причины создания:**
- Корни атаковали спящих — погибло ~30 000 Айни
- Кошмары Хола хлынули с окраин
- Стало ясно: нужны те, кто будет бодрствовать

**Первый Командор:** Ворн Неспящий — выживший во время Войны, который не смог заснуть после пережитого ужаса. Он превратил своё «проклятие» в призвание.

### Эволюция

| Период              | Изменения                                          |
| ------------------- | -------------------------------------------------- |
| ~800–600 лет назад  | Формирование традиций, первые ритуалы              |
| ~600–400 лет назад  | Расширение, строительство застав                   |
| ~400–100 лет назад  | Период мира, снижение численности                  |
| Последние 100 лет   | Стабильность, ~1500 воинов                         |
| Последние 5 лет     | Рост нагрузки из-за усиления Кошмаров              |

## Обет Бодрствования

Центральный элемент ордена — **Обет Бодрствования**: клятва не спать во время Зимнего Покоя.

### Суть обета

> *«Пока народ спит, я бодрствую. Пока Древо грезит, я стерегу. Мой сон — жертва. Моя бессонница — щит.»*
> — Клятва Бодрствующих

**Что означает обет:**
- Отказ от участия в священном Зимнем Покое
- Разрыв связи с коллективным сном Хола
- Использование зелий и ритуалов для поддержания бодрствования
- Постепенное истощение тела и разума

### Цена обета

Каждый год без полноценной спячки **сокращает жизнь на ~2 года**.

| Срок службы      | Состояние                                    |
| ---------------- | -------------------------------------------- |
| 1–5 лет          | Небольшое истощение, адаптация               |
| 5–10 лет         | Хроническая усталость, ранняя седина         |
| 10–15 лет        | Серьёзное истощение, проблемы со здоровьем   |
| 15–20 лет        | Критическое состояние, большинство умирает   |
| 20+ лет          | Единицы выживают, легенды ордена             |

**Средняя продолжительность службы:** 15–20 лет
**Средний возраст смерти Бодрствующего:** 45–50 лет

### Зелье Бодрствования

Специальный отвар, позволяющий не спать.

**Состав:**
- Сок глубинных корней (запретный компонент)
- Экстракт Цветов Сновидений
- Мёд Хола
- Секретные травы, известные только аптекарям ордена

**Эффект:**
- Подавляет потребность в сне
- Поддерживает физические силы
- Вызывает лёгкую эйфорию
- При длительном применении — привыкание и истощение

### Командование

```
            ┌─────────────────────┐
            │     КОМАНДОР        │
            │   (Грим Несонный)   │
            └─────────┬───────────┘
                      │

      │               │               │

└─────┬─────┘   └─────┬─────┘   └─────┬─────┘
      │               │               │
   Стражи          Погра-          Морская
   Залов           ничники         Стража
   (~800)          (~400)          (~200)
                      │
                   Охотники
                   (~100)
```

### Подразделения

| Подразделение      | Численность | Функции                                    |
| ------------------ | ----------- | ------------------------------------------ |
| **Стражи Залов**   | ~800        | Охрана Корневых Залов изнутри              |
| **Пограничники**   | ~400        | Патрулирование границ с Гнилыми Топями     |
| **Морская Стража** | ~200        | Охрана порта и побережья                   |
| **Охотники**       | ~100        | Уничтожение Кошмаров Хола (элита)          |

### Командор: Грим Несонный

| Параметр          | Значение                                           |
| ----------------- | -------------------------------------------------- |
| **Возраст**       | 58 лет                                             |
| **Служба**        | 40 лет в Ордене, 12 лет как Командор               |
| **Рекорд**        | Не спал 43 Зимних Покоя — абсолютный рекорд        |
| **Прозвище**      | «Вечно Бодрствующий», «Тень без сна»               |

**Внешность:**
- Седые волосы (поседел в 30)
- Глубоко запавшие глаза с тёмными кругами
- Худощавое, жилистое тело
- Движения экономные, точные
- Почти никогда не моргает

**Характер:**
- Суровый, немногословный
- Фанатично предан защите народа
- Не доверяет иностранцам
- Физически истощён, но не признаёт слабости
- Видит вещи, которые другие не видят (возможно, галлюцинации)

**Слабость:** Приближается к пределу. Тело отказывает, но воля несгибаема. Следующий Зимний Покой может стать последним.

## Ритуалы и традиции

### Ритуал Посвящения

**Испытание Трёх Ночей:**
1. Кандидат проводит три ночи в Гнилых Топях
2. Без сна, без помощи, с минимальным снаряжением
3. Должен выжить и не сойти с ума
4. Если вернулся в здравом рассудке — принят

**Смертность на испытании:** ~20%

### Ритуал Прощания

Перед каждым Зимним Покоем Бодрствующие прощаются с семьями.

**Церемония:**
- Семья благословляет воина
- Обмен амулетами (на случай смерти)
- Клятва верности («Я вернусь или останусь на страже вечно»)
- Последний совместный ужин

### Ритуал Последнего Сна

Когда Бодрствующий чувствует приближение конца, он может выбрать **Последний Сон**:
- Отказ от Зелья Бодрствования
- Погружение в сон — впервые за годы или десятилетия
- Смерть во сне — душа наконец присоединяется к Хола

Это считается **почётной смертью**, а не самоубийством.

### Зал Неспящих

Мемориал в Заставе Рассвета.

**Содержимое:**
- Имена всех погибших Бодрствующих
- Личные вещи легендарных командоров
- Амулеты тех, кто не вернулся с испытания
- Вечно горящий огонь (символ бодрствования)

## Статус в обществе

### Почитание

Бодрствующие — **одни из самых уважаемых людей** в Торсии.

**Причины почитания:**
- Жертвуют сном (священным действом) ради народа
- Защищают беззащитных во время спячки
- Противостоят Кошмарам Хола
- Живут короткую, но героическую жизнь

### Привилегии

| Привилегия                    | Описание                                    |
| ----------------------------- | ------------------------------------------- |
| Содержание семьи              | Государство обеспечивает семью воина        |
| Почётное захоронение          | У корней Хола (высшая честь)                |
| Освобождение от податей       | Для всей семьи                              |
| Право на реликвии             | Фрагменты коры-летописи                     |
| Доступ к запретным зонам      | Для выполнения долга                        |

### Отношение народа

- Матери молятся, чтобы их дети НЕ стали Бодрствующими
- Но если стали — гордятся безмерно
- Бодрствующего угощают в любой таверне бесплатно
- Отказ помочь Бодрствующему — позор

## Враги ордена

### Кошмары Хола

Главные противники — существа из Гнилых Топей.

| Тварь              | Опасность | Тактика охоты                        |
| ------------------ | --------- | ------------------------------------ |
| Теневые волки      | Средняя   | Работа в группе, факелы              |
| Гнилые медведи     | Высокая   | Копья, засады, численное превосходство |
| Шепчущие           | Очень высокая | Затычки в уши, янтарные амулеты   |
| Корневые черви     | Низкая    | Огонь, ловушки                       |

### Внешние угрозы

- Потенциальные захватчики (не было веками)
- Шпионы [[Империя Углеас|Углеаса]] (подозрения)
- Контрабандисты (редко)

### Внутренние угрозы

- Адепты [[Путь Корня|Пути Корня]] (расследования)
- Дезертиры (крайне редко, величайший позор)

## Текущее состояние

### Кризис

Последние 5 лет — **тяжелейший период** в истории ордена:
- Кошмары Хола стали агрессивнее и многочисленнее
- Гнилые Топи расширяются
- Потери выросли на 30%
- Набор новобранцев не покрывает убыль

### Проблемы

| Проблема                    | Последствия                            |
| --------------------------- | -------------------------------------- |
| Нехватка личного состава    | Перегрузка оставшихся                  |
| Истощение командора         | Кризис лидерства на горизонте          |
| Усиление Кошмаров           | Больше потерь, меньше сна (ирония)     |
| Подозрения о Пути Корня     | Внутреннее напряжение                  |

### Позиция фракции «Ястребов»

Бодрствующие — основа фракции **«Стражей Грани»**:
- Требуют активных действий против Гнилых Топей
- Хотят расширения ордена
- Считают пассивность Совета Жрецов опасной
- Готовы к радикальным мерам
