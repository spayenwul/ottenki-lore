# География Торсии

Физическая география, биомы, поселения и экономические ресурсы [[Торсия|Торсии]].

---

## Общие данные

| Параметр       | Значение                           |
| -------------- | ---------------------------------- |
| **Площадь**    | ~40 000 км²                        |
| **Тип**        | Континентальное государство        |
| **Население**  | ~318 000 человек                   |
| **Поселений**  | 1 город + ~40 деревень и застав    |
| **Плотность**  | ~8 чел/км² (низкая)                |

## Расположение

Торсия расположена в [[Внешнее кольцо|Внешнем кольце]], на восточной его части:
- **Север:** [[Великое Герцогство Сэл]] (дом [[Зефирель, Шёпот Шторма|Зефирели]])
- **Юг:** [[Великое Герцогство Олсия]] (дом [[Сервелос, Стремительная Тень|Сервелоса]])
- **Запад:** [[Море Жизни|Внутреннее море]] (отделяет от [[Империя Углеас|Империи Углеас]])
- **Восток:** [[Внешний океан]] (открытые воды)

**Положение на континенте:**
Торсия занимает относительно узкую полосу земли между двумя Великими Герцогствами. [[Великое Древо Хол]] возвышается в центре территории, его корни пронизывают всю страну от границы до границы.

**Порядок государств Внешнего кольца** (по часовой стрелке от юго-запада):
> Теократия Бролез → Схабди → Элзия → Королевство Хакирия → Империя Ареан → Великое Герцогство Сэл → **Торсия** → Великое Герцогство Олсия → Графство Геун → южный ледник

    │    │  ┌───────────────┐    │        │

◄───│    │  │     ХОЛ       │    │        │───►
    │    │  │   (центр)     │    │        │ Внешний
Внут│    │  └───────────────┘    │        │ океан
мор │    │                       │        │
    │    │     ★ КОРИЛЬ          │        │
    │    │     (восточное        │        │
    │    │      побережье)       │        │

## Биомы и экосистемы

### Зона Кроны («Сердце Древа»)

Территория непосредственно под кроной [[Великое Древо Хол|Хола]] и вокруг ствола.

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Климат**         | Вечное тёплое лето, ~25°C круглый год             |
| **Освещение**      | Золотистый свет, листья светятся ночью            |
| **Особенности**    | Целебный воздух, ускоренный рост растений         |
| **Доступ**         | Только жрецы и паломники                          |

**Уникальная флора:**
- **Плоды Хола** — золотистые фрукты с целебными и эйфорическими свойствами
- **Цветы Сновидений** — белые цветы, пыльца которых вызывает пророческие видения
- **Мох Памяти** — покрывает кору, впитывает и хранит воспоминания

**Уникальная фауна — «Дети Хола»:**
- **Золотопёрые птицы** — разносят семена, их помёт — лучшее удобрение
- **Медовые светляки** — освещают ночную крону, их свет исцеляет раны
- **Хранители Коры** — жуки размером с кулак, защищают кору от паразитов

### Внутренние земли («Земли Корней»)

Основная зона проживания [[Айни]], где корни Древа пронизывают почву.

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Климат**         | Мягкий умеренный, тёплые зимы                     |
| **Почва**          | «Древесная почва» — смесь перегноя и корневых волокон |
| **Особенности**    | Высокая плодородность, стабильные урожаи          |
| **Население**      | ~280 000 (88% населения)                          |

**Сельское хозяйство:**
- Клубневые (особые сорта, адаптированные к Древесной почве)
- Фруктовые сады
- Пчеловодство (мёд с особыми свойствами)
- Скотоводство (небольшие стада)

**Особенности экосистемы:**
- Почва реагирует на сильные эмоции и магию
- Растения «знают» о приближении Зимнего Покоя раньше, чем Айни
- Корневые волокна пронизывают всю землю — Древо «чувствует» каждый шаг

### Пограничные леса («Дикие Земли»)

Лесные территории на границах Торсии с соседними государствами.

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Климат**         | Умеренный континентальный, выраженные сезоны      |
| **Растительность** | Густые смешанные леса, подлесок                   |
| **Особенности**    | Охотничьи угодья, добыча древесины, пограничные заставы |
| **Население**      | ~30 000 (охотники, лесорубы, пограничники)        |

**Ресурсы:**
- Древесина (обычные деревья, не от Хола)
- Дичь (олени, кабаны, птица)
- Лекарственные травы
- Янтарь Древа (собирают ближе к центру)

**Фауна:**
- Рыжегривые медведи — священные животные, спутники некоторых кланов
- Лесные волки
- Дикие кабаны
- Олени

### Корневые Залы («Сердце Сна»)

Подземная сеть убежищ среди живых корней Древа, раскинувшаяся под всей территорией страны.

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Расположение**   | Под всей территорией, связаны туннелями           |
| **Климат**         | Постоянные +18°C, высокая влажность               |
| **Особенности**    | Стены — живые корни, источающие тепло             |
| **Вместимость**    | ~400 000 спящих (с запасом)                       |

**Структура:**
- **Верхние залы** — для простых Айни, тёплые и безопасные
- **Средние залы** — для жрецов, ближе к «снам» Древа
- **Глубинные залы** — запретная зона, граница с Тёмным Сном

**Уникальные явления:**
- Корни светятся мягким золотистым светом
- Воздух насыщен успокаивающими спорами
- Сны спящих переплетаются — коллективное сновидение

### Гнилые Топи («Кошмарные Земли»)

Мёртвые зоны на юго-западе страны, где влияние [[НПС/Лиан]] просачивается напрямую.

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Расположение**   | Юго-западная часть Торсии                         |
| **Климат**         | Постоянный туман, холод, отсутствие солнца        |
| **Особенности**    | Чёрная вода, искажённая флора, Кошмары Хола       |
| **Статус**         | Запретная территория, табу                        |

**Признаки:**
- Вода чёрная и маслянистая, но не пахнет гнилью — пахнет **пустотой**
- Деревья мертвы, но не падают — застыли в агонии
- Туман поглощает звуки
- Длительное пребывание вызывает кошмары наяву

**«Кошмары Хола» (фауна Топей):**
- **Теневые волки** — безглазые хищники, охотятся на звук
- **Гнилые медведи** — искажённые версии рыжегривых, агрессивные и безумные
- **Шепчущие** — гуманоидные силуэты из тумана, сводят с ума шёпотом
- **Корневые черви** — паразиты, атакующие корни Древа

## Побережья

### Восточное побережье (Внешний океан)

Главное побережье Торсии, выходящее на [[Внешний океан]].

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Тип**            | Скалистые утёсы с редкими бухтами                 |
| **Главный порт**   | [[Кориль]] — столица и единственный крупный порт  |
| **Особенности**    | Рыболовство, торговля с внешним миром             |

**Значение:**
- Единственный выход во внешний мир для торговли
- Связь с [[Империя Углеас|Углеасом]] и другими государствами
- Защищён скалами и сложной навигацией

### Западное побережье (Внутреннее море)

Побережье, выходящее на [[Море Жизни|Внутреннее море]].

| Параметр           | Характеристики                                    |
| ------------------ | ------------------------------------------------- |
| **Тип**            | Низменное, болотистое                             |
| **Порты**          | Нет крупных (мелкие рыбацкие деревни)             |
| **Особенности**    | Заболоченность, мало пригодно для торговли        |

**Значение:**
- Рыболовство (пресноводная рыба)
- Добыча торфа
- Граничит с Гнилыми Топями на юге

## Поселения

### Кориль — столица

| Параметр         | Значение                                          |
| ---------------- | ------------------------------------------------- |
| **Население**    | ~48 000                                           |
| **Расположение** | Восточное побережье, крупнейшая бухта             |
| **Особенности**  | Единственный порт, торговый и политический центр  |
| **Основан**      | ~600 лет назад                                    |

**Описание:** Крупнейшее поселение Торсии, построенное в защищённой бухте на восточном побережье. Здания из белого камня и живой древесины (корни Хола достигают даже сюда). Храм Толкователей возвышается в центре города.

**Районы:**
- **Гавань** — порт, склады, торговые ряды, Квартал Чужеземцев
- **Сердцевина** — административный центр, Храм Толкователей
- **Корневые Кварталы** — жилые районы
- **Ремесленный Круг** — мастерские, кузницы, ткачи

**Особенности:**
- Единственное место, где разрешено присутствие иностранцев
- Не впадает в полную спячку — небольшой гарнизон бодрствует круглый год
- Корни Древа достигают города через подземные туннели

### Малые поселения

| Поселение           | Население | Особенность                                    |
| ------------------- | --------- | ---------------------------------------------- |
| **Сильван**         | ~8 000    | Центр деревообработки, северные леса           |
| **Медовая Роща**    | ~6 000    | Пчеловодство, сады, внутренние земли           |
| **Верхний Корень**  | ~5 000    | Ближайшее к стволу, жреческий центр            |
| **Застава Рассвета**| ~3 000    | Пограничный форт на границе с Гнилыми Топями   |
| **Рыбачья Коса**    | ~4 000    | Рыболовецкая деревня, восточное побережье      |
| **Янтарный Холм**   | ~3 500    | Добыча янтаря, центральные земли               |

**Общее число деревень:** ~40
**Деревенское население:** ~270 000

**Самодостаточность:**
- Торсия почти полностью самодостаточна в продовольствии
- Импорт ограничен: металлы, ткани, предметы роскоши

## Транспорт и пути сообщения

### Морской транспорт
- **Внешняя торговля:** Через порт Кориля на Внешний океан
- **Каботаж:** Вдоль восточного побережья
- **Сезонность:** Порт закрыт во время Зимнего Покоя

### Корневые пути
- Сеть туннелей среди корней Хола
- Пронизывают всю территорию страны
- Используются для перемещения во время подготовки к спячке
- Некоторые участки запретны (ведут к Глубинным Залам)

### Сухопутные дороги
- Главный тракт: Кориль — Верхний Корень — северная граница
- Грунтовые дороги между деревнями
- Качество хорошее благодаря стабильному климату
- Зимой не используются (народ спит)

### Пограничные переходы

| Направление | Переход              | Примечание                           |
| ----------- | -------------------- | ------------------------------------ |
| На север    | Застава Северных Ветвей | Граница с Великим Герцогством Сэл  |
| На юг       | Застава Южного Листа | Граница с Великим Герцогством Олсия  |
| На запад    | (нет)                | Гнилые Топи и болота                 |

## Влияние географии на культуру

### Положение между соседями
- Торсия зажата между двумя Великими Герцогствами
- Исторически — буферная зона
- Изоляционизм частично вынужденный — проще не вмешиваться в чужие дела
- Соседи уважают Хола и не посягают на территорию

### Древо как центр мира
- Вся жизнь вращается вокруг Хола
- Направления определяются относительно Древа («к стволу», «от ствола»)
- Социальный статус связан с близостью к Древу
- Жрецы живут ближе всего, изгои — на границах

### Гнилые Топи как табу
- Формируют страх перед «изнанкой» божества
- Создают культуру молчания о тёмных сторонах веры
- [[Путь Корня]] черпает силу из этого страха
- Изгнание в Топи — худшее наказание (хуже смерти)

### Сезонный ритм
- Три сезона: Пробуждение (весна), Труд (лето-осень), Покой (зима)
- Никакого индивидуального планирования — всё подчинено коллективному циклу
- Спячка не воспринимается как потеря времени, а как **другая форма жизни**
