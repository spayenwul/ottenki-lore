# Магия Торсии

Магические традиции [[Торсия|Торсии]], черпающие силу из связи с [[Великое Древо Хол|Великим Древом Хол]].

---

## Общая характеристика

Магия [[Айни]] уникальна — она основана не на манипуляции внешними силами, а на **симбиотической связи с живым божеством**. Маги Торсии не приказывают, а **просят**, не берут, а **получают**.

### Особенности

| Аспект           | Характеристика                                           |
| ---------------- | -------------------------------------------------------- |
| **Источник**     | [[Великое Древо Хол]] (через сны и корни)                |
| **Стиль**        | Медленный, ритуальный, требует медитации                 |
| **Сила**         | На территории Торсии — значительная; вне — слабеет       |
| **Ограничения**  | Зависимость от воли Хола, невозможность агрессии против Древа |

### Связь с Древом

Все маги Торсии получают силу через **сны**:
- Во время обычного сна — слабая связь
- Во время Зимнего Покоя — сильная связь
- В трансе — прямой контакт с сознанием Хола

**Важно:** Хол может отказать. Если Древо «не одобряет» действие мага, заклинание не сработает.

## Школы магии

### Магия Роста (Зеленосплетение)

Управление растениями и жизненной энергией.

**Возможности:**

| Заклинание           | Описание                                          |
| -------------------- | ------------------------------------------------- |
| Ускорение роста      | Растение за минуты проживает недели               |
| Корневой барьер      | Стена из сплетённых корней и лиан                 |
| Живой мост           | Временный мост из переплетённых ветвей            |
| Шипованная плеть     | Лиана с шипами как оружие                         |
| Укоренение           | Обездвиживание цели корнями из-под земли          |

**Применение:**
- Сельское хозяйство (ускорение урожая)
- Строительство (выращивание структур)
- Оборона (барьеры, ловушки)
- Боевое применение (редко)

**Ограничения:**
- Работает только с живыми растениями
- На территории Торсии — сильнее
- Вдали от Древа — требует семена или саженцы

### Магия Исцеления (Древесное Благословение)

Целительство, основанное на жизненной силе Хола.

**Возможности:**

| Заклинание            | Описание                                         |
| --------------------- | ------------------------------------------------ |
| Малое исцеление       | Заживление порезов, ушибов                       |
| Очищение              | Нейтрализация ядов и болезней                    |
| Восстановление        | Лечение серьёзных ран (требует время)            |
| Регенерация           | Восстановление утраченных частей тела (ритуал)   |
| Дыхание жизни         | Возвращение только что умершего (крайне редко)   |

**Особенность:** Целители используют **Плоды Хола** как фокус. Без плода — магия слабее.

**Известные целители:**
- Ольха Мягкорукая (Хранитель Ветви Исцеления)
- Клан Тихой Воды (специализируется на целительстве)

### Магия Защиты (Коревое Укрытие)

Создание барьеров и защитных чар.

**Возможности:**

| Заклинание            | Описание                                         |
| --------------------- | ------------------------------------------------ |
| Щит Листа             | Персональный барьер из уплотнённого воздуха      |
| Коревой Панцирь       | Кора нарастает на теле как броня                 |
| Убежище               | Быстрое укрытие из выросших ветвей               |
| Круг Защиты           | Барьер вокруг группы                             |
| Благословение Хола    | Долгосрочная защита от болезней и слабых атак    |

**Применение:**
- Защита караванов и экспедиций
- Укрепление [[Корневые Залы|Корневых Залов]]
- Защита во время опасных ритуалов

### Магия Снов (Грёзоплетение)

Самая уникальная и потенциально опасная школа Торсии.

**Возможности:**

| Заклинание            | Описание                                         |
| --------------------- | ------------------------------------------------ |
| Чтение снов           | Просмотр чужих снов (с согласия)                 |
| Послание во сне       | Отправка сообщения спящему                       |
| Пророческий транс     | Видения будущего (ненадёжные)                    |
| Направление сна       | Изменение хода чужого сна                        |
| Совместный сон        | Несколько человек видят один сон                 |

**Применение:**
- Толкование пророчеств
- Связь между жрецами
- Терапия кошмаров
- Обучение (передача знаний через сон)

**Опасности:**
- Можно «заблудиться» в чужом сне
- Прикосновение к снам Хола — риск безумия
- Граница с [[#Запретная магия (Путь Корня)|запретной магией]] размыта

## Запретная магия (Путь Корня)

Магия, черпающая силу из **«Тёмного Сна»** [[Великое Древо Хол|Хола]] — энергии [[НПС/Лиан]].

### Официальная позиция

**Запрещена категорически.**

Использование карается изгнанием в [[Гнилые Топи]] и лишением права на спячку (фактически — смерть).

### Возможности (по слухам)

| Заклинание            | Описание                                         |
| --------------------- | ------------------------------------------------ |
| Наведение кошмара     | Насылание ужаса на спящего                       |
| Теневой удар          | Атака чистой тьмой                               |
| Контроль Кошмара      | Подчинение существ Гнилых Топей                  |
| Иллюзия страха        | Создание видений худших страхов цели             |
| Питие Тени            | Поглощение энергии Лиан напрямую (смертельно опасно) |

### Адепты Пути Корня

Последователи [[Путь Корня|запретного учения]] практикуют эту магию тайно.

**Признаки адепта:**
- Бессонница или отказ от Зимнего Покоя
- Частые кошмары у окружающих
- Притяжение к Гнилым Топям
- Глаза с тёмными разводами (на поздних стадиях)

### Опасности

| Риск                   | Последствия                                     |
| ---------------------- | ----------------------------------------------- |
| Безумие                | Потеря связи с реальностью                      |
| Порабощение            | Воля [[НПС/Лиан]] может поглотить разум              |
| Трансформация          | Физические мутации (в худших случаях)           |
| «Пробуждение» Хола     | Если ритуал слишком мощный — катастрофа для всех|

## Магические артефакты

### Янтарь Древа

Застывшая смола [[Великое Древо Хол|Хола]] — главный магический ресурс Торсии.

**Свойства:**
- Хранит магическую энергию
- Усиливает заклинания роста и исцеления
- Светится при активации
- Защищает от мелкой нечисти

**Применение:**

| Форма                 | Использование                                   |
| --------------------- | ----------------------------------------------- |
| Необработанный        | Фокус для заклинаний                            |
| Амулет                | Защита от кошмаров                              |
| Светильник            | Вечный источник света                           |
| Линза                 | Усиление магии снов                             |
| Оружие (наконечник)   | Эффективно против Кошмаров Хола                 |

### Плоды Хола

Священные плоды — не просто еда, а **магический инструмент**.

**Применение:**
- Фокус для целительной магии
- Ингредиент для зелий
- Вызывает пророческие видения (при правильном употреблении)
- Усиливает связь с Древом

**Ограничения:**
- Нельзя выносить за пределы Торсии (портятся)
- Собирают только жрецы
- Злоупотребление вызывает зависимость

### Кора-летопись

Фрагменты коры с узорами — **инструменты прорицания**.

**Применение:**
- Толкование будущего по узорам
- Запись пророчеств
- Медитация (созерцание узоров)

### Другие артефакты

| Артефакт              | Свойства                                        |
| --------------------- | ----------------------------------------------- |
| Чаша Снов             | Усиливает магию сновидений                      |
| Посох Хранителя       | Усиливает защитную магию                        |
| Зелье Бодрствования   | Позволяет не спать ([[Орден Бодрствующих]])     |
| Факел Покоя           | Отпугивает Кошмаров                             |

## Обучение магии

### Кто может стать магом

В Торсии **нет чёткого разделения** на магов и обычных людей. Каждый Айни имеет связь с Хола, просто разной силы.

**Уровни способностей:**

| Уровень           | Доля населения | Возможности                          |
| ----------------- | -------------- | ------------------------------------ |
| Обычный           | ~70%           | Базовые ритуалы, благословения       |
| Одарённый         | ~25%           | Простые заклинания одной школы       |
| Жрец              | ~4%            | Полноценная магия 1-2 школ           |
| Высший жрец       | ~1%            | Мастерство во всех школах            |

### Путь обучения

1. **Детство:** Замечают вещие сны, сильную связь с природой
2. **Отбор (10-12 лет):** Жрецы проверяют способности
3. **Обучение (10 лет):** При храме, под руководством наставника
4. **Первое Погружение:** Ритуальный транс у глубоких корней
5. **Служение:** Работа на благо общины

### Храм Толкователей как академия

Главный центр обучения магии — **Храм Толкователей** в [[Кориль|Кориле]].

**Структура обучения:**
- Младшие послушники (базовые ритуалы)
- Ученики (специализация в одной школе)
- Служители (практикующие маги)
- Слушатели Снов (мастера)

## Ограничения и особенности

### Зависимость от Хола

| Фактор                | Влияние на магию                               |
| --------------------- | ---------------------------------------------- |
| Расстояние от Древа   | Чем дальше — тем слабее                        |
| Состояние Хола        | Если Древо «болеет» — магия нестабильна        |
| Одобрение Древа       | Хол может отказать в силе                      |
| Зимний Покой          | Во сне — магия сильнее; бодрствуя — слабее     |

### За пределами Торсии

Магия Айни **резко слабеет** вдали от Торсии:
- На расстоянии 100 км — сила падает вдвое
- На расстоянии 500 км — только базовые заклинания
- На расстоянии 1000+ км — почти полная потеря способностей

**Исключение:** Маг с крупным янтарным артефактом сохраняет часть силы.

### Конфликт с другими традициями

| Традиция              | Взаимодействие                                 |
| --------------------- | ---------------------------------------------- |
| [[Плетение]] (Углеас) | Несовместимы; янтарь мешает Плетению           |
| Магия Бролез          | Нейтрально; разные источники                   |
| Демоническая магия    | Конфликт; Хол отторгает демоническую энергию   |

## Аномалии и феномены

### Искажение прорицаний

На территории Торсии любая магия предсказания **ненадёжна**:
- Видения смешиваются со снами Хола
- Результаты двусмысленны или ложны
- Даже опытные провидцы ошибаются

**Причина:** Постоянная борьба Светлого и Тёмного Снов создаёт «помехи».

### Коллективные видения

Иногда во время Зимнего Покоя весь народ видит **один и тот же сон**:
- Считается посланием от Хола
- Жрецы толкуют его как пророчество
- Может быть предупреждением об угрозе

### Кошмары наяву

В Гнилых Топях возможны **галлюцинации** даже без магии:
- Туман сам по себе вызывает видения
- Долгое пребывание ведёт к безумию
- Защита — янтарные амулеты и Зелье Ясности
