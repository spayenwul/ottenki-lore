# TL;DR — Школы магии Теократии Бролез

> 5 школ, пронизанных религиозным фанатизмом и памятью о мёртвой богине Серафиэль.

- **Живущая Душа** — расширение заклинаний на всю армию
- **Хор Живой Души** — манипуляция верой (вдохновение союзников, деморализация врагов)
- **Морозные Заветы** — лёд + вера; бонус урона по демонам
- **Культ Истреблённого Света** — еретическая школа без религиозных ограничений
- **Хоровод Падших** — манипуляция душами мёртвых через фрагменты тела Серафиэль
