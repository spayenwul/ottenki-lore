# Школа Морозных Заветов

**Государство:** [[Теократия Бролез]]
**Тип:** Контроль / Урон по демонам
**Статус:** Официальная доктрина Теократии

## Описание

Эта школа развилась благодаря суровому климату северных владений Теократии. Её магия соединяет веру с энергией ледников — маги видят в вечном льде застывшую благодать Серафиды, нетленную и чистую.

## Философия

Лёд — это чистота, остановившая время. Он не гниёт, не разлагается, не поддаётся демонической скверне. Истинная вера подобна льду: она сковывает зло и сохраняет жизнь.

## Плетение

Медленные, выверенные движения. Маги словно лепят что-то из воздуха — их жесты нарочито сдержанны, почти ритуальны. Чрезмерная поспешность разрушает структуру ледяного плетения.

## Заклинания

### Щит Вечного Холода
Создаёт ледяной барьер, блокирующий атаки. Перед магом вырастает кристаллическая стена льда толщиной в несколько локтей — она выдерживает несколько мощных ударов, прежде чем рассыпается осколками.

### Оковы Легаррона
Замораживает землю вокруг, сковывая врагов. Маг ударяет ногой — морозная волна расходится по земле, покрывая её льдом и замораживая ноги всех, кто стоит в радиусе заклинания.

### Ледяной Кинжал Праведника
Магический клинок, который наносит огромный урон демоническим созданиям. Маг формирует клинок из чистого льда, насыщённого верой — для людей это просто холод, для демонов — жгучая боль, разрушающая их субстанцию.
