## IV. Фракции

### Евклиды: Дом Солнечного Горизонта + Культ Единого Света

| Файл | Содержание |
|------|-----------|
| [[Дом Солнечного Горизонта]] | Патриарх Иларион, Жреческая Гвардия |
| [[Писание Единого Света]] | Идеология Культа: доктрина, ритуалы |

**Цель:** Доминация Евклидов. «Очищение». Захват Центрального Острова.
**Силы:** ~24 000 (Гвардия 1300 + ополчение 8000 + Культ 15 000). Магия утроена.
**Скрытая правда:** Культ создан и управляется [[Ковен Живущих во Смерти#Вейра Солнцеедка — Жрица Дисбаланса|Вейрой/Сэйрин]] (Ковен).

### Залиды: Клан Багровой Матери + Красные Горизонты

| Файл | Содержание |
|------|-----------|
| [[Южный берег Врат-над-Морем влияние Клана Багровой Матери]] | Территория и влияние Клана |
| [[Школа Багровой Матери]] | Магическая традиция (кровь, земля) |

**Цель:** Выживание. Месть за Красную Гавань.
**Силы:** ~12 000. Магия ослаблена на 30-40%.
**Проблема:** Красная Гавань в руинах — экономическая база уничтожена.

### Нейтралы: Орден Двойного Пламени

| Файл | Содержание |
|------|-----------|
| [[Орден Двойного Пламени]] | Орден: история, структура, кризис |
| [[Академия Сияющей Земли]] | Последний нейтральный оплот |
| [[Тейас Межсолнечный]] | Ректор-полукровка, ранен. Имеет Писание Ксвима |

**Цель:** Сохранить единство. Удержать Врата.
**Силы:** ~50 магов 3-4 круга.
**Проблема:** Тейас ранен. Орден раскалывается. Два предателя внутри (Солан Карей → Углеас, Шелла Вир'Хас → Ковен).

### Империя Углеас

| Файл | Содержание |
|------|-----------|
| [[Империя Углеас]] | Обзор Империи |
| [[История Империи Углеас]] | 92 года к контролю над Вратами |

**Цель:** Протекторат над Вратами. Контроль торговли.
**Силы:** 5 000 в анклаве + интервенционный корпус.
**Метод:** «Гуманитарное вмешательство» через 2-3 недели (ускорено из-за Аламии).

### Гильдия Морских Клинков

**Цель:** Прибыль. Адмирал Велиар продаёт услуги обеим сторонам.
**Силы:** ~3 500. Флот. Русалы.
**Инфильтрация:** [[Ковен Живущих во Смерти#Сивилла Зернистая — Жница|Сивилла]] (Ковен) на уровне логистики.

### Скрытые игроки

**Ковен Живущих во Смерти:**

| Файл | Содержание |
|------|-----------|
| [[Ковен Живущих во Смерти]] | Полное досье: Стражи Колыбели, план Пробуждения, 4 ячейки |

**Цель:** Пробудить кость [[Серафиэль 1|Серафиэль]] внутри Врат.
**Агенты:** Эол (в стене Врат 110 лет), Сивилла (торговый магнат), Кассиан (подводный), Вейра = Сэйрин.
**Статус:** Ускоряют работу — Агнейр далеко от Врат.

**Элизабет Шоттиль:**

| Файл | Содержание |
|------|-----------|
| [[Элизабет Шоттиль Эрбиан Юстий]] | Досье принцессы |
| [[Сюжет/Концепты/Элизабет и Ковен — тихий зонтик]] | Обнаружила Ковен → не мешает → прикрывает |

**Цель:** Уничтожить Врата → монополия Вторых Врат.
**Статус:** Портал Аламии = «мягкая подушка» для разрушения. Решимость растёт.

**Демоны-подстрекатели:**

| Файл | Содержание |
|------|-----------|
| [[Персонажи/Демоны/Вейлан Рэй]] | Обскур Домена Близнецов, ~140 лет, 4-5 личин в городе |

**Метод:** Ложные флаги, провокации в обличье обеих сторон.
**Вейлан лично:** Философ перемены. Летит к башне к +20ч.

## IX. Схема взаимодействия

```
                    УГЛЕАС (интервенция 2-3 нед.)
                         │
                         │ хочет контроль
                         ▼
    ┌─── ЭЛИЗАБЕТ ───► БЛОКИРУЕТ ◄─── Агенты Красного Крыла
    │    (тихий зонтик)     │
    │         │              │
    │         │ не даёт      │ не даёт
    │         ▼              ▼
    │    АГНЕЙР         ОБНАРУЖЕНИЕ
    │  (на БАШНЕ —      (заглушено)
    │   далеко от            │
    │   Врат!)               ▼
    │              ┌── ВРАТА ЖИЗНИ ──┐
    │              │   рунная сеть   │
    │              │    слабеет      │
    │              └────────┬────────┘
    │                       │
    │         ┌─────────────┼─────────────┐
    │         ▼             ▼             ▼
    │    ЕВКЛИДЫ       ЗАЛИДЫ        КОВЕН
    │    (Культ=       (оборона)   (ускоряют —
    │     Вейра)                   Агнейр далеко)
    │         │                         │
    │         └──── ВОЙНА ──────────────┘
    │              (топливо для
    │               Пробуждения)
    │
    └──── ВЫГОДА: канал мёртв → Вторые Врата = монополия

         АЛАМИЯ ─── портал через БАШНЮ ─── Рахат-Маар
              (беженцы уходят, город слабеет,
               но меньше жертв)
```
