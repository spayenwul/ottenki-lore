## III. Фракции и их планы

### Евклиды: Дом Солнечного Горизонта + Культ Единого Света

| Файл | Содержание |
|------|-----------|
| [[Дом Солнечного Горизонта]] | Фракция Евклидов: Патриарх Иларион, Жреческая Гвардия |
| [[Писание Единого Света]] | Идеология Культа: доктрина, ритуалы |
| [[Сцена — Эдрис и Сейрин]] | Сцена: разведчик Агнейра встречает Пророчицу |

**Цель:** Доминация Евклидов. «Очищение» от Залидов. Захват Центрального Острова.
**Силы:** ~24 000 (Гвардия 1300 + ополчение 8000 + Культ 15 000). Магия утроена.
**Скрытая правда:** Культ создан и управляется [[Ковен Живущих во Смерти#Вейра Солнцеедка — Жрица Дисбаланса|Вейрой/Сэйрин]] (Ковен).

### Залиды: Клан Багровой Матери + Красные Горизонты

| Файл | Содержание |
|------|-----------|
| [[Южный берег Врат-над-Морем влияние Клана Багровой Матери]] | Территория и влияние Клана |
| [[Школа Багровой Матери]] | Магическая традиция Залидов (кровь, земля) |

**Цель:** Выживание. Месть за Красную Гавань. Удержание южных территорий.
**Силы:** ~12 000 (Алые Стражи 1200 + ополчение 6000 + Красные Горизонты 2000 + беженцы 3000). Магия ослаблена на 30-40%.
**Проблема:** Экономическая база уничтожена (Красная Гавань в руинах).

### Нейтралы: Орден Двойного Пламени

| Файл | Содержание |
|------|-----------|
| [[Орден Двойного Пламени]] | Орден: история, структура, кризис |
| [[Академия Сияющей Земли]] | Академия как последний нейтральный оплот |
| [[Тейас Межсолнечный]] | Ректор-полукровка, ранен, ослаблен |

**Цель:** Сохранить единство. Удержать Академию и Врата.
**Силы:** ~50 магов 3-4 круга. Стража Ордена (ослаблена расколом).
**Проблема:** Тейас ранен. Орден раскалывается по расовому признаку.

### Внешняя сила: Империя Углеас

| Файл | Содержание |
|------|-----------|
| [[Империя Углеас]] | Обзор Империи |
| [[История Империи Углеас]] | История и интересы в регионе |

**Цель:** Протекторат над Вратами. Контроль торгового маршрута.
**Силы:** 5 000 солдат в анклаве. Интервенционный корпус на подходе.
**Метод:** Наблюдение → накопление компромата → «гуманитарное вмешательство».
**Роль в покушении:** Обеспечили линзу-артефакт, не предотвратили заговор.

### Нейтралы-циники: Гильдия Морских Клинков

**Цель:** Прибыль. Выживание. Адмирал Велиар готовит бегство, но продаёт услуги обеим сторонам.
**Силы:** ~3 500. Флот. Альянс с Русалами (подводный контроль). Сеть Мелеефатов.
**Особенность:** Инфильтрированы [[Ковен Живущих во Смерти#Сивилла Зернистая — Жница|Сивиллой]] (Ковен) на уровне логистики.

## IV. Скрытые игроки

### Ковен Живущих во Смерти — «Колыбель Воскрешения»

| Файл | Содержание |
|------|-----------|
| [[Ковен Живущих во Смерти]] | Полное досье: идеология, структура, **Стражи Колыбели**, план Пробуждения |

**Цель:** Пробудить кость Серафиэль внутри Врат. Трансформировать Врата в первый орган Тела богини.
**Силы:** 4 Стража-Кости (бессмертные маги 140-190 лет) + 60 Плоти + 200 Крови + 50 000 Культа (невольно).
**Метод:** Война ослабляет рунную сеть → порог 30% → финальный штурм Культа → ритуал Пробуждения.
**Таймлайн:** 2-3 недели до финального штурма. 1 ночь на ритуал.
**Ключевые агенты:**
- **Эол** — замурован в стене Врат 110 лет, проведёт ритуал
- **Сивилла** — «Мадам Тасс», торговый магнат, 80 лет в городе
- **Кассиан** — подводный хищник, 60+ мертвецов под Вратами
- **Вейра** — она же Пророчица Сэйрин, создательница войны

### Элизабет Шоттиль — «Тихий зонтик»

| Файл | Содержание |
|------|-----------|
| [[Элизабет Шоттиль Эрбиан Юстий]] | Досье принцессы |
| [[Сюжет/Концепты/Элизабет и Ковен — тихий зонтик]] | Стратегия Элизабет: обнаружение Ковена, решение не мешать, прикрытие операции |
| [[Империя Ареан]] | Контекст Ареанской империи |

**Цель:** Ослабить Углеас и Элзию. Монополия через Вторые Врата.
**Метод:** Обнаружила операцию Ковена. Не мешает — прикрывает. Блокирует тех, кто может помешать (особенно Агнейра).
**Силы:** Агенты Красного Крыла + Дом Эребус + Дракх'Мегас.
**Роль:** Невидимый щит Ковена. Партия столкнётся с её блокадой, не понимая причин.

### Агнейр Делир — Архитектор (неосознанная угроза)

| Файл | Содержание |
|------|-----------|
| [[Агнейр Делир]] | Игровой персонаж: демон-архитектор Врат |
| [[АГНЕЙР ДЕЛИР ЛИЧНОСТЬ, ПАМЯТЬ, ВОЗВРАЩЕНИЕ]] | Его связь с городом и Вратами |
| [[Ритуал связи]] | Механика связи Агнейра с конструкцией |

**Роль:** Единственный, кто может обнаружить зародыш и остановить Пробуждение.
**Проблема:** Отвлечён Элизабет (через Дракх'Мегаса), войной, детьми, личными проблемами.
**Если вернётся к Вратам:** Услышит кость. Может деактивировать сеть управляемо ИЛИ (если убедить) ускорить Пробуждение.

## VIII. Схема взаимодействия сторон

```
                    УГЛЕАС (интервенция)
                         │
                         │ хочет контроль
                         ▼
    ┌─── ЭЛИЗАБЕТ ───► БЛОКИРУЕТ ◄─── Агенты Красного Крыла
    │    (тихий зонтик)     │
    │         │              │
    │         │ не даёт      │ не даёт
    │         ▼              ▼
    │    АГНЕЙР         ОБНАРУЖЕНИЕ
    │    (отвлечён)      (заглушено)
    │                        │
    │                        ▼
    │              ┌── ВРАТА ЖИЗНИ ──┐
    │              │   рунная сеть   │
    │              │    слабеет      │
    │              └────────┬────────┘
    │                       │
    │         ┌─────────────┼─────────────┐
    │         ▼             ▼             ▼
    │    ЕВКЛИДЫ       ЗАЛИДЫ        КОВЕН
    │    (Культ=       (оборона)     (Стражи)
    │     Вейра)                        │
    │         │                         │
    │         └──── ВОЙНА ──────────────┘
    │              (топливо для
    │               Пробуждения)
    │
    └──── ВЫГОДА: канал мёртв → Вторые Врата = монополия
```
